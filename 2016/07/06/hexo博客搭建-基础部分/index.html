hexo博客搭建 --- 基础部分

Veröffentlicht am 2016-07-06

Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other languages) and Hexo generates static files with a beautiful theme in seconds.

搭建博客所需工具

  • Node.js
  • Git

##初始化博客
通过使用如下命令可初始化自己的博客:

其中<folder>是你博客的路径，如～/myBlog/。当然我们也可以先cd到博客myBlog文件夹中然后再使用$ hexo init命令初始化博客。
现在博客已经初始化完成！我们可以先通过$ hexo generate或$ hexo g命令生成博客的静态文件，然后再通过$ hexo server或$ hexo s命令启动本地博客服务。在浏览器中输入 http://localhost:4000便可查看我们搭建好的博客。

为博客更换主题

搭建好的博客具有如下结构：

├── _config.yml
├── package.json
├── scaffolds
├── source
 |      ├── _drafts
 |      └── _posts
└── themes

其中themes文件夹中存放了博客的主题

如图所示，landscape和next就是两个不同的主题，以安装next主题为例，我们首先将主题下载到themes文件夹中，然后打开博客根目录下的配置文件_config.yml，将’theme‘一项的landscape替换为next就可以了。
接下来我们通过$ hexo clean命令清空博客缓存，$ hexo g命令重新生成博客静态文件以及$ hexo s命令启动本地服务。可以看到，我们的博客以及应用了新的主题。
