CCF试题——有趣的数

发表于 2016-09-06   |   分类于 算法   |     |   阅读次数

题目

问题描述

我们把一个数称为有趣的，当且仅当：

  1. 它的数字只包含0, 1, 2, 3，且这四个数字都出现过至少一次。
  2. 所有的0都出现在所有的1之前，而所有的2都出现在所有的3之前。
  3. 最高位数字不为0。

因此，符合我们定义的最小的有趣的数是2013。除此以外，4位的有趣的数还有两个：2031和2301。
请计算恰好有n位的有趣的数的个数。由于答案可能非常大，只需要输出答案除以1000000007的余数。

输入格式

输入只有一行，包括恰好一个正整数n (4 ≤ n ≤ 1000)。

输出格式

输出只有一行，包括恰好n 位的整数中有趣的数的个数除以1000000007的余数。

样例输入

4

样例输出

3

思路

根据题目中0在1之前、2在3之前以及最高位不为0我们可以推出2一定会出现在首位。考虑一个有趣的数的前k位只使用了部分数的情况，我们可以发现状态集中一共有6种情况：

状态 使用的数字 未使用的数字
0 2 0 1 3
1 2 0 1 3
2 2 3 0 1
3 2 0 1 3
4 2 3 0 1
5 2 0 1 3 无

注意
只有以上6种情况满足题目约束，以使用数字2、1 未使用数字3、0为例，前k位中有1，后n-k位中一定会存在0与题目约束0在1之前不符，故排除。其它情况皆如此。

容易看出，在前k位使用部分数字的条件下的不同排列可以由在前k-1位使用部分数字的条件下的排列添加1个数字得到。以state[k][i]表示某个有趣的数的前k位是第i种状态的不同排列的数量，我们可以得到：

  • $state[k][0] = 1 $

    显然前k位只用2的情况只有1种

  • $state[k][1] = state[k-1][0] + state[k-1][1]*2$

    前k位只有2和0的状态可由前k-1位只由2的状态加一个0得到，也可以由前k-1位已经包含了2和0的状态再加上一个2或0得到。

  • $state[k][2]=state[k-1][0]+state[k-1][2]$

    前k位只有2和3的状态可由前k-1位只由2的状态加一个3得到，也可以由前k-1位已经包含了2和3的状态再加上一个3得到。这里因为2不能在3的后面故比上一个状态少了一种情况。

  • $state[k][3]=state[k-1][1]+state[k-1][3]*2$

    前k位只有2、0和1的状态可由前k-1位只由2和0的状态加一个1得到，也可以由前k-1位已经包含了2、0和1的状态再加上一个2或1得到。

  • $state[k][4]=state[k-1][1]+state[k-1][2]+state[k-1][4]*2$

    前k位只有2、3和0的状态可由前k-1位只由2和0的状态加一个3得到，也可由前k-1位只由2和3的状态加一个0得到，亦可由前k-1位已经包含了2、3和0的状态再加上一个3或0得到。

  • $state[k][5]=state[k-1][3]+state[k-1][4]+state[k-1][5]*2$

    前k位包含了2、3、0、1的状态（最终态）可由前k-1位只由2、3和0的状态加一个1得到，也可由前k-1位只由2、0和1的状态加一个3得到，亦可由前k-1位已经包含了2、3、0、1的状态再加上一个1或3得到。
    最终输出的答案：$state[n][5]$
